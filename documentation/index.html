Abstract

An ontology for modelling a JAMS file containing different types of musical annotations. In particular, this ontology focuses on the Chord Annotations contained in the JAMS file and their attributes.

Music Time Index Comonentc back to ToC or Class ToC

IRI: http://w3id.org/polifonia/ontology/jams/MusicTimeIndexComponent

A component of a MusicTimeIndex, which defines the value of the component and its ValueType. Multiple MusicTimeIndexComponents are required for representing points on a sequence that requires more than one value to be defined. For example, a symbolic notation usually needs beats and measures to define a point on a sequence: in this case there will be a MusicTimeIndexComponents defining the measure position and another MusicTimeIndexComponents for defining the beat position.
